Portfolio

Applikationer vi har utvecklat.

NocKnoc

System för att vid ytterdörren presentera alla anställda på företaget på en surfplatta. När en person valdes kontaktades personen och personens arbetsteam via Slack.

Tekniker: Flutter, Dart, Javascript, Node.js, Slack API.

Mobilapplikation

Sociala medier applikation med funktionalitet att skapa och redigera personlig profil samt chatfunktionalitet byggt med Firebase.

Tekniker: React Native, Firebase Autentication, Firebase Realtime Database, Javascript.

Profilbilder som används i bilden ovan ägs av respektive ägare. Ingen upphovsrättsintrång är avsedd.
Tidrapporteringssystem

En digitaliseringslösning för företaget BEHAB:s tidigare manuella tidsrapporteringssystem. Tidsrapporteringsystemet tar hand om varje anställds arbetade tider och vilka maskiner och verktyg som används under den arbetade tiden. Systemet sammanställer veckorapporter som sedan används för bokföring.

Tekniker: React, Material-UI, MongoDB, Javascript, Node.js, Python, Flask, Google Sheet API .

"Vi har fått bättre kontroll över vårt arbete. Vi har redan sparat mycket pengar, på tider vi tidigare missade att fakturera"

VD, Bockara Entreprenad och Handel Aktiebolag
